## JavasScript - ES10 & ES11

### Trim Start & Trim End

The trimStart() method removes spaces from the beginning of a string.

```
Ex: '      Hello World      '

Result: 'Hello World      '
```

### Try / Catch

Try catch error parameter is optional.

```
Example 1: 

try {
  code ...
} catch(error)  {
  return error;
}

Example 2:

try {
  code ...
} catch  {
  code ...
}
```
